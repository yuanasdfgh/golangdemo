# 移动端项目

## 项目创建
``` 
npx degit dcloudio/uni-preset-vue#vite-ts my-vue3-project
```
## 引入三方库

```
pnpm add pinia pinia-plugin-persist-uni @dcloudio/uni-ui

pnpm add -D miniprogram-api-typings @types/node

pnpm add -D unplugin-vue-components unplugin-auto-import #自动引入

pnpm add -D postcss-px-to-viewport-8-plugin # px转vw/vh

```
